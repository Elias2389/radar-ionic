<strong>Google maps Radar</strong>
===================

Description
-------------

Google maps Radar with Ionic Framework:

> **Functions:**
> - Get geolocaton.
> - Get places into radar and add markers of each place.

> **Requeriments:**
> - Nodejs.
> - Ionic Framework.
> - ngx-translate. 
> - Google maps native plugins.
> - Geolocation native plugins.
> - Nearbysearch API.
